1)BASICS

Info:
	whatis xxx
		Displays info on xxx comand
	info xxx
		Displays info on xxx comand
	xxx --help
		Displays info on xxx comand
	man xxx
		Displays info on xxx comand
	which -a xxx
		Displays location of xxx command
	file xxx
		Displays info of xxx file or directory

	cat /proc/cpuinfo 
		Displays info about cpu cores

Users:
	su xxx
		Switchs to user xxx provided you give the password
		
				
2)FILES

Basics
	mkdir -p grandpa/dad/me
		Creates a directory me and required ancestors
	rmdir
		Remove empty dir
	cat xxx
		Shows content of file	
	tail/head -10 xxx
		Shows ten last/first lines of the file
	tar -zxvf yourfile.tar.gz 
	    Unzip tar files

	md5sum filename
		To check integrity using md5 
		
Mounting:
	sudo gedit /etc/fstab

	Add a line like: /dev/sdax /media/user/label   ntfs  defaults       0  0 
	
	df -h [.]
		Shows info on partitions and mounting points. "." to show partition of current directory

Bashing:
	Alias to Bash
		edit /etc/bash.bashrc
			add to the end: alias myalias='originalcommand' (eg alias py='python'). '' when using parameters
	Default bash set in...
		/ect/passwd
	To reload
		exec bash

Path
	export PATH=$PATH:/opt/acroread/bin
		Add paths to $PATH
	ln -s targetfile linkname
		To make a link to a file
		
Grep
	grep xxx /usr/share/dict/words
		Find matches of string xxx (words is a mini dictionary!)
			
Permissions
	chmod u+x, 
		u,g,o +,- r,w,x
